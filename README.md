"# University Exam Management System" 
"" 
"## ?? Project Description" 
"A complete university exam management system with automatic scheduling, role-based access control, and PDF export functionality." 
"" 
"## ? Features" 
"- ?? JWT Authentication with role-based access (Admin, Professor, Student)" 
"- ?? Automatic exam scheduling with conflict prevention" 
"- ?? Department-based organization (GI & IDS)" 
"- ?? PDF export of exam schedules for students, professors, and admins" 
"- ?? Room and supervisor management" 
"- ?? Module organization by semester (S1-S6)" 
"- ?? User management with role-specific dashboards" 
"" 
"## ??? Technologies Used" 
"### Frontend" 
"- React.js" 
"- Tailwind CSS" 
"- jsPDF & jspdf-autotable" 
"- Axios" 
"" 
"### Backend" 
"- Node.js" 
"- Express.js" 
"- MongoDB with Mongoose" 
"- JWT for authentication" 
"- bcrypt for password hashing" 
"" 
"## ?? Installation" 
"" 
"### Prerequisites" 
"- Node.js (v14 or higher)" 
"- MongoDB (local or Atlas)" 
"" 
"### Backend Setup" 
"\`\`\`bash" 
"cd backend" 
"npm install" 
"npm run dev"

"" 
"### Frontend Setup" 
"\`\`\`bash"
